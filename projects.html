Projects & Case Studies

Selected work, framed by outcome

Representative automation initiatives drawn from my roles — each shown as the problem, my approach, the tooling, and the impact.

Framework

Regression Automation Framework in Selenium + TestNG

Problem

Manual regression cycles were slow and error-prone, delaying releases and letting UI defects slip through.

Approach

Designed a Page Object Model framework with reusable components and TestNG data providers, then scheduled nightly runs so regressions surfaced automatically.

Tools

Selenium WebDriverTestNGPOMMavenJava
Impact: Cut regression effort dramatically and gave the team repeatable, cross-browser coverage on every build.
Data Quality

API & Data-Warehouse Validation with SQL

Problem

MicroStrategy reports and downstream APIs needed to be trusted, but data discrepancies were only caught late — in production.

Approach

Wrote layered SQL checks comparing warehouse outputs to source systems and validated REST/SOAP responses in Postman, catching mismatches at the data layer.

Tools

SQLMicroStrategyREST / SOAPPostmanJMeter
Impact: Caught data defects before release and raised confidence in business-critical reporting.
BDD + CI

BDD Test Suite with Cucumber + Jenkins CI

Problem

QA, developers, and business stakeholders lacked a shared, readable definition of "done" for each feature.

Approach

Authored Gherkin scenarios mapping requirements to executable steps, then integrated the suite into Jenkins with clear pass/fail reporting on every commit.

Tools

CucumberGherkinSeleniumJenkinsJIRA
Impact: Aligned the whole team on behavior and delivered fast, automated feedback in the pipeline.
